# Legislative Bill Analyst

An AI-powered tool for Senate interns and staffers working the Commerce, Technology, and AI portfolio. Paste any Congress.gov bill URL or select from your saved bills — get an instant analytical summary, policy brief, internal memo, and vote recommendation, all generated by Claude.

Built as a standalone HTML/CSS/JS app. No build tools, no frameworks, no backend required.

---

## What it does

- **Fetch any bill** from Congress.gov by pasting its URL
- **Bill library** — pre-loaded with recent Commerce, Tech, and AI bills from the 119th Congress; add your own
- **Four AI-generated outputs** per bill:
  - Analytical summary with stakeholder landscape
  - One-page policy brief with formal section headers
  - Internal Senate memo formatted for staff use
  - Vote recommendation (Aye / Nay / Abstain) with rationale
- **Copy or export** any output as a `.txt` file
- **Responsive** — works on desktop and mobile
- **Dark mode** supported automatically

## Setup

### 1. Get an Anthropic API key

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Create an account and navigate to **API Keys**
3. Click **Create Key** and copy it

### 2. Open the app

Download or clone this repository, then open `index.html` in any modern browser. No install, no server needed.

```bash
git clone https://github.com/YOUR_USERNAME/legislative-analyst.git
cd legislative-analyst
open index.html   # macOS
# or double-click index.html in Windows/Linux
```

### 3. Enter your API key

When the app first loads, it will prompt you for your API key. Paste it in. The key is saved in your browser's `localStorage` — it stays on your machine and is sent directly to Anthropic's API only when you generate analysis.

> **Security note:** Never commit your API key to GitHub. This app stores it only in your browser, not in any file. If you fork this repo, the `.gitignore` already excludes any `.env` files.

## Adding your own bills

Open `js/bills.js` and add entries to the `SAMPLE_BILLS` array:

```javascript
{
  id: "S.1234",
  title: "Your Bill Title",
  sponsor: "Sen. Smith (D-NY)",
  status: "In committee",
  topic: "ai",          // one of: ai, tech, commerce, telecom
  date: "Jun 2025",
  summary: "Brief description of what the bill does.",
  url: "https://www.congress.gov/bill/119th-congress/senate-bill/1234"
}
```

Or use the **Fetch from Congress.gov** field in the sidebar — paste a Congress.gov URL and the app will look up the bill automatically.

## Congress.gov API (optional upgrade)

The current fetch feature uses Claude's web search to look up bills. For higher volume or more structured data, you can connect directly to the [Congress.gov API](https://api.congress.gov/):

1. Request a free API key at [api.congress.gov](https://api.congress.gov/)
2. See `js/app.js` for where to add direct API calls

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your app will be live at `https://YOUR_USERNAME.github.io/legislative-analyst`

Users will enter their own API key when they open the app.

## Usage notes

- This tool is designed for **research and drafting assistance**. Always verify bill text, status, and sponsor information directly on [congress.gov](https://www.congress.gov) before using outputs in official communications.
- Generated memos, briefs, and vote recommendations are a **starting point** — they should be reviewed and edited by the staffer before use.
- API usage is billed to your Anthropic account. Each generation (summary, brief, memo, vote rec) uses approximately 500–1,500 tokens.
